Tool 4 / 5 — Hero-Sprite aus 8x8-Rasterzellen zusammensetzen

Erwartet hero.png im selben Ordner. Klick auf eine Rasterzelle = erste Ecke, Klick auf eine zweite Zelle = zweite Ecke → die Auswahl ist das Rechteck aus allen 8x8-Zellen dazwischen. Dann Richtung + Frame wählen und auf "+ Hinzufügen" klicken (oder direkt in der Tabelle auf die Zielzelle klicken). Ein neuer Klick im Bild startet eine neue Auswahl.

Klick = 1. Ecke wählen
⚠️ hero.png nicht gefunden. Datei danebenlegen oder IMG_SRC im Script anpassen.

Aktuelle Auswahl (8x8-Raster)

Pixel: 0,0 – 16x32

Zuweisen: Richtung × Frame

Alternative: direkt in der Tabelle auf eine Zelle klicken speichert die aktuelle Auswahl dort ebenfalls. Spalten = Frame (0=Schritt-links,1=Stand,2=Schritt-rechts), Zeilen = Richtung.
